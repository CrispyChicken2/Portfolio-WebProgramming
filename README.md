# Portfolio-WebProgramming
Create your own portfolio from scracth with a colleague

## Description
Ce projet est un ePortfolio simple développé avec FastAPI.  
Il permet d'afficher une liste de projets et d'en ajouter via une interface administrateur.

---

## Fonctionnalités
- Affichage des projets sur une page d’accueil
- Page admin pour ajouter des projets
- Gestion des formulaires avec FastAPI
- Rendu dynamique avec Jinja2
- Redirection après ajout d’un projet

---

## Technologies utilisées
- Python
- FastAPI
- Jinja2
- HTML
